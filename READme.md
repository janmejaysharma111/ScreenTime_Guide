******App_Name******
Screen Guide

******Purpose******
this app monitors your screen and analyzes content to categorize your screen time and guide to reduce manage screen time and also advices if high amount of distractive content is watched and manages quality of contnent watched to analyze and predict productivity

*****Contents******
1. A monitor that reads the screen and finds out the total daily and weekly screen time.
2. Catgoriezes the content watched on the screen into study, skills, games, entertainment etc.
3. Evaluates productivity and gives tips for management.
4. Allows to make 'Blockers' for certain content of two types
    i. completely turn off the tab/app when the targeted content is detected.
    ii. convince the user to turn off the app by user suggested method and instructions
5. Screentime management chatbot and Goal Streak setting option

*****Tech_Stack******
- running terminal commands through pythoncode
- chatbot using api
- tkinter to make the app
- locally storing the screen data for the day/week 
- Machine learning to categorize content into fields (entertainment,skills,study etc)
